# 📤 Manuel Deploy (Vercel Dashboard)

## Adım 1: Projeyi Zip'le

1. `bs3dcrafts` klasörünü sağ tıkla
2. "Sıkıştır" veya "Send to > Compressed folder"
3. `bs3dcrafts.zip` oluştur

## Adım 2: Vercel'e Yükle

1. https://vercel.com adresine git
2. "Add New..." > "Project"
3. "Import Git Repository" yerine
4. "Deploy from CLI or API" seç
5. Veya direkt dosya yükle (drag & drop)

## NOT

Bu yöntem pek önerilmez çünkü:
- Her değişiklikte manuel yükleme gerekir
- Git history kaybolur
- Otomatik deploy çalışmaz

**Daha iyi:** Vercel CLI kullanın (HIZLI_DEPLOY.bat)
