## 🐛 Sorun Varsa

### Console Hatası
1. F12 tuşuna basın
2. Console sekmesine gidin
3. Kırmızı hata var mı kontrol edin
4. Varsa bana gönderin

### Network Hatası
1. F12 → Network sekmesi
2. Sayfayı yenileyin
3. Kırmızı (failed) request var mı?
4. Varsa tıklayıp detayları gönderin

### Database Hatası
Eğer "Database error" görürseniz:
- Vercel'de DATABASE_URL kontrol edin
- Supabase'de connection pooling aktif mi kontrol edin

### JWT Hatası
Eğer "JWT error" görürseniz:
- Vercel'de JWT_SECRET var mı kontrol edin
- Redeploy yapın
